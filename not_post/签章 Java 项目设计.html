3 业务逻辑

根据 Model 划分业务

划分 Order， Contract， Ca， Transaction 和 Log 模块

Bussiness 作为业务模块， 业务逻辑 组合调用 Order， Contract， Ca， Transaction 和 Log 模块内容

技术选型

  • 后端框架： Spring Boot
  • 业务数据库： Mongo
  • 缓存： Redis
  • 持久层: MongoRepository
  • 消息队列：RabbitMq
  • 文档生成工具：Swagger2
  • 日志组件：slf4j
  • Java 实用工具： Lombok
  • 测试框架：JUnit5， MockMvc
  • 其他：
    • 日志记录：Spring AOP
